## 📄 プロジェクト概要

このリポジトリは、プロダクト **mimamori** の開発に関するドキュメントとコードを管理しています。

---

## 📂 ドキュメント構成

主要なドキュメントは、プロジェクト全体に関するものと、特定のプロダクト **mimamori** に関するものに分かれて格納されています。

### 1. プロジェクト共通ドキュメント (`docs/`)

プロジェクト全体の計画、要件、タスクなどが格納されています。

* **`docs/design.md`**: プロジェクトの**設計**に関するドキュメント。
* **`docs/requirements.md`**: プロジェクトの**要件**に関するドキュメント。
* **`docs/tasks.md`**: 現在**やるべきこと**（タスクリスト）をまとめたドキュメント。
* **`docs/wants.md`**: プロジェクトに対する**要求**（望ましい機能など）をまとめたドキュメント。

### 2. プロダクトドキュメント (`mimamori/docs/`)

プロダクト **mimamori** に特化したドキュメントが格納されています。

* **`mimamori/docs/`**: プロダクト **mimamori** に関するその他のドキュメント。

---

## 💻 プロダクトディレクトリ

* **`mimamori/`**: プロダクト **mimamori** の**ソースコード**や関連ファイルが格納されているディレクトリ。
